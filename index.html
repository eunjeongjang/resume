카카오페이 인증
Overview

  • 본 문서는 (주)카카오페이의 기밀 자산으로 카카오페이 인증 API(이하 "인증 API") 개발과 관련된 표준 가이드와 필요한 정보를 제공하기 위하여 작성되었습니다.
  • (주)카카오페이의 승인 없이 이를 변형하거나 제 3 자에게 제공 및 배포 할 수 없습니다. 또한 (주)카카오페이의 사전 승인 없이 "인증 API"를 재가공하거나 재판매하는 등의 활동은 할 수 없습니다.
  • (주)카카오페이는 이 문서를 작성하면서 현재의 정확하고 완전한 정보를 제공하기 위해 노력하였으나, 부득이하게 오류나 누락이 있을 수 있습니다. 이와 같은 건에 대해 알려주시면 빠른 시일 내에 반영 및 업데이트 하겠습니다.

1. 들어가는 말

1.1. 목적
본 문서는 카카오페이 인증 서비스를 소개합니다.
1.2. 대상
본 문서는 카카오페이 인증 서비스 전반에 대해 알고 싶은 분에게 적합합니다.
1.3. 기술적인 가이드에 대해
기술적인 가이드에 대해서는 본 문서와 함께 제공되는 카카오페이 인증 연동 공통사항 문서 및 각 서비스코드에 해당하는 문서를 참고하시기 바랍니다.
  • 카카오페이 인증 연동 공통사항 문서는 API 연동 전 알아두셔야 할 내용에 대해 다루고 있습니다.
  • 서비스코드 문서에서는 해당 서비스의 구현을 돕는 API에 대해 다루고 있습니다.

2. 카카오페이 인증에
대하여

  • 카카오페이에서 제공하는 인증 서비스를 소개합니다.
  • 카카오페이 인증은 카카오톡 기반의 사설인증 API 상품입니다.
  • 사용자에 대해 전자 서명 혹은 본인확인이 필요한 경우 카카오페이 인증서로 서명하여 인증을 수행할 수 있습니다.
  • 서명 완료 후 생성된 전자문서는 CMS Signed Data로, PKCS#7 syntax를 기반으로 하고 있습니다.
    • 보다 상세한 내용을 원하실 경우 카카오페이 인증 - 전자문서 및 UID 검증 가이드 (카카오인증_4.Appendix-signedData.pdf) 문서를 참고하시기 바랍니다.
  • 카카오페이에서 제공하는 인증서는 다음과 같은 특징을 가지고 있습니다.
    • PKI 국제표준 전자서명 기술을 이용하며, X.509.V3 스펙을 따르고 있습니다.
    • Block-Chain을 활용하여 인증서 키를 관리합니다.
    • 금융보안원 보안적합성 시험 수행을 완료했습니다.
  • 두 가지 인증 방식을 제공합니다.
    • App to App 이용기관의 앱에서 custom scheme을 이용하여 카카오톡을 호출하여 인증을 진행합니다.
    • Talk Message 이용기관 서버에서 카카오페이 서버로 인증 요청을 보내면, 카카오페이에서 사용자의 카카오톡으로 Talk Message를 보내 인증을 진행합니다.

3. 제공 서비스

카카오페이 인증은 아래와 같은 경우에 사용할 수 있습니다.

service code, description 항목을가진 제공서비스 리스트입니다.
Service Code Description
S110, S120 자동이체 출금동의 증빙자료 생성
S210, S220 각종 계약서에 대한 전자서명
S211 우편물 수령확인
S310, S311, S315, S320 간편인증
S510 개인정보가 있는 모바일웹 열람
S610, S620 간편로그인
S621 자동서명을 지원하는 간편로그인

상기 서비스의 세부 내용은 각 서비스코드별 문서에서 확인 가능합니다.

  • 1. 이용기관 서버에서 카카오페이 인증 서버로 전자서명을 요청합니다.
  • 2. 사용자는 카카오톡을 이용하여 전자서명을 진행합니다.
    • 사용자에게 전자서명 내용을 전달합니다.
    • 사용자는 카카오톡 앱 내에서 전자서명을 수행합니다.
    • 카카오페이 인증 서버로 서명 결과를 전송합니다.
  • 3. 1번 단계 이후, 이용기관 서버는 카카오페이 인증에서 제공하는 전자서명 상태조회 API를 통해 서명 완료 여부를 조회할 수 있습니다.
  • 4. 전자서명이 완료되면 서명값 검증 API를 통해 검증을 수행합니다.
    • 서명값 검증 API에 대한 응답으로 서명 완료 후 생성된 전자문서(CMS Signed Data)를 받을 수 있습니다.
    • 전자문서에 대한 자세한 내용이 궁금하신 경우, 카카오페이 인증 - 전자문서 및 UID 검증 가이드 (카카오인증_4.Appendix-signedData.pdf) 문서를 참고하시기 바랍니다.
  • 5. 서명값 검증까지 성공적으로 완료되면 카카오페이 인증 프로세스가 완료됩니다.
